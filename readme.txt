To run the web app,create javaWebApplication on Netbeans ,put the folders and files existing in the web folder in the web folder in NetBeans,
 use Apache Tom Cat as a server and connect to the data base. then click on run icon, it will open our website.
There is a text box, write your query in it, then click enter, it will open another page contains results .
The results is Titles to the pages contain your query, when yo click on one of them ,it will open the page corresponding to it.
If your query is not found, it will open the page contains "Sorry,Your query is not found".
To go back to the home page ,you can click on the top left image or you can use back arrow on browser
